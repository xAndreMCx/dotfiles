# Dotfiles

This repository contains personal configuration files for a Linux environment, primarily optimized for **Arch Linux**. It utilizes a modular approach with profiles to switch between hardware configurations and color schemes.

---

## Tech Stack

* **Window Manager:** Hyprland (Wayland Compositor)
* **Shell:** Fish Shell with Starship prompt
* **Terminal:** WezTerm
* **Editor:** Neovim
* **Dotfile Manager:** Bombadil
* **File Manager:** Yazi
* **Launcher:** Rofi
* **Notifications:** Dunst

---

## Dependencies

Ensure the following font is installed for icons to render correctly:

* `ttf-material-symbols-variable-git`

---

## Installation and Profile Management

This setup uses **Bombadil** to manage symlinks and templates.

### Profiles
The configuration supports hardware-specific settings and themes:

* **Hardware:** `desktop` (NVIDIA drivers, dual-monitor setup) or `laptop` (touchpad optimizations)
* **Themes:** `catppuccin_mocha` or `tokyo_night`
